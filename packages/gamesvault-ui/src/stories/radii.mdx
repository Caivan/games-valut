## Usage

- Use `none` for sharp corners or when a rectangular shape is desired.  
- Use `sm` and `md` for subtle rounding on buttons, inputs, and smaller components.  
- Use `lg` and `xl` for cards, modals, and containers requiring more softness.  
- Use `round` for fully circular elements like avatars or icon buttons.
