## Usage Summary

- **none**: No spacing, for elements that must touch directly.  
- **xs**: Very tight padding/margin, e.g., space between icon and label inside buttons.  
- **sm**: Small gaps like margin between form inputs or spacing in tight layouts.  
- **md**: Default padding around cards, containers, or section gaps.  
- **lg**: Larger margins between major UI sections or groups of controls.  
- **xl and above**: Spacious layouts, big sections, and whitespace for breathing room.
